# Check IRS EIN

A Raycast extension to validate Employer Identification Numbers (EINs) against the IRS database to verify if they belong to nonprofit organizations.

## Features

This extension provides two ways to check EINs:

1. **Check EIN (with argument)**: Directly enter an EIN as an argument when running the command
2. **Search EIN (from clipboard)**: Copy an EIN to your clipboard and run this command to validate it

## How to Use

### Check EIN (with argument)
1. Open Raycast
2. Type "Check EIN"
3. Enter the 9-digit EIN when prompted
4. View the validation result

### Search EIN (from clipboard)
1. Copy a 9-digit EIN to your clipboard
2. Open Raycast
3. Type "Search EIN"
4. View the validation result

## Requirements

- Raycast v1.95.0 or higher
- Internet connection to access the IRS API

## Development

This extension is built with TypeScript and the Raycast API.
